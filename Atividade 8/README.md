# Atividade 8 - Java

## Descrição
Este repositório contém diversas atividades de programação em Java, organizadas em diferentes diretórios. Cada atividade possui classes e arquivos-fonte relacionados, abordando conceitos como herança, polimorfismo, interface e manipulação de objetos.

## Estrutura do Projeto
O projeto está dividido em diretórios, cada um contendo uma atividade específica:

- **`Atividade_1`**: Classes `Produto` e `Main` para manipulação de produtos.
- **`Atividade_2`**: Versão aprimorada de `Produto` com funcionalidades adicionais.
- **`Atividade_3`**: Hierarquia de classes `Funcionario`, `Gerente` e `Desenvolvedor`.
- **`Atividade_4`**: Continuação da hierarquia de `Funcionario` com mais funcionalidades.
- **`Atividade_5`**: Implementação de uma interface `IMeioTransporte` com classes como `Carro`, `Bicicleta` e `Trem`.
- **`Atividade_6`**: Hierarquia de classes de animais (`Animal`, `Cachorro`, `Gato`, `Vaca`).
- **`Atividade_7`**: Implementação de diferentes formas de pagamento (`Pix`, `Boleto`, `CartaoCredito`).
- **`Atividade_8`**: Empresa com diferentes tipos de funcionários (`Gerente`, `Desenvolvedor`, `Estagiario`).

## Pré-requisitos
- Java JDK (versão 11 ou superior)
- IDE de sua preferência (IntelliJ IDEA, Eclipse, VS Code)
- Git (opcional, para controle de versão)

## Como Executar Localmente
1. **Clone o Repositório**
   ```bash
   git clone <URL_DO_REPOSITORIO>
   ```
   
2. **Importe o Projeto na IDE**
   - Abra sua IDE e importe o projeto como um projeto Java existente.
   - Certifique-se de que todas as dependências estejam corretamente configuradas.

3. **Compile e Execute**
   - Navegue até o diretório da atividade que deseja executar.
   - Certifique-se de que o diretório contém a classe `Main.java`.
   - Abra a classe `Main.java` correspondente.
   - Compile e execute o código diretamente pela IDE ou utilizando o terminal:
     ```bash
     javac Main.java
     java Main
     ```

## Informações Adicionais
- As classes `Main.java` de cada atividade contêm exemplos de uso e demonstrações dos conceitos abordados.
- O projeto está configurado para ser utilizado em uma estrutura de módulos (`.iml`), o que facilita a organização e execução em uma IDE como IntelliJ IDEA.
